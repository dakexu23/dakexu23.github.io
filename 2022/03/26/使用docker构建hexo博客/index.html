使用docker构建hexo博客

发表于 2022-03-26 更新于 2022-03-28 分类于 docker

实现目标

由于疫情隔离在家，想要做一件事就是把自己一些知识和经验整理成经验。有如下的一些想法“

  1. 技术类相关的文档以hexo博客的形式呈现；
  2. 本地启动docker容器，并进行博客撰写。通过编辑Markdown文件，然后使用hexo渲染成静态页面；
  3. 静态页面build至nginx镜像；
  4. 本人使用的是Mac M1芯片的电脑，目前发现build出来的镜像无法在x86架构下的服务器运行，所以通过阿里云的镜像仓库服务自动构建（代码需要提交至github）。
  5. 由于博客已经通过Git提交至github，可以使用github提供的静态页面服务，以自己的用户名生成username.github.io的博客；
  6. 在个人的云主机上通过拉取镜像发布个人的博客；
